About this website

This site was developed by Tim Hickey as a demo for the course he is teaching this semester, CS10a: Introduction to Problem Solving Using Python. It is implemented using Flask with Jinja templates and Twitter Bootstrap styling, using Chart.js for the plots. The source code is available at https://github.com/tjhickey724/covid19demo

The data comes from a the covidtracking project www.covidtracking.com which collects data on testing and covid19 deaths from all states and many US territories.

Our next step is to extend this to cover world data from all states using the timeseries data collected at https://github.com/pomber/covid19 Here is a direct link to the timeseries data: https://pomber.github.io/covid19/timeseries.json

These time series only change once a day so a better approach would be to just run a chron job to produce static files for each of the possible menu choices, but the load should be relatively light and the code is pretty efficient. I currently have the system only check every 5 minutes for new data.
